# PlyPaint
Графический редактор на С++ для курсовой (6 семестр, САПР, МВС-17, Компьютерная графика)

Задание №18.
1. Создание графического редактора.
a) Построение объектов: отрезок, полилиния (состоящая из чередующихся отрезков и дуг, построенных по трем точкам), окружность, дуга.
b) Удаление выбранных объектов. 
c) Вывод и редактирование текстов с возможностью выбора типа и размера шрифтов, угла поворота текста и места размещения на экране. Строка текста рассматривается как объект.
Выбор объектов производить тремя способами:
a) выбор объекта (группы объектов) мышью; 
b) выбор с использованием прямоугольной рамки: 
c) выбранными считаются те, которые полностью попали внутрь прямоугольной области; 
d) выбранными считаются и те, части которых попали внутрь прямоугольной области.
2. Организация диалога с помощью меню, причем построение и удаление объектов должно быть представлено в виде режимов работы, а не в виде отдельных операций. Ввод объектов производить в режиме “тянущаяся линия”.
3. Сохранение и считывание результатов работы в файлах.
